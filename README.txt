Fast Tennis Clash - GitHub Pages Version
==============================================

ملفات اللعبة جاهزة للرفع على GitHub Pages.

خطوات الرفع:
1. سجل دخولك على GitHub وعمل repository جديد Public باسم: fast-tennis-clash
2. ادخل على المستودع → Add file → Upload files → اختر كل الملفات من هذا المجلد
3. Commit changes
4. افتح Settings → Pages → Source → اختر Branch main و Folder / (root) → Save
5. بعد دقيقة أو دقيقتين، اللعبة تعمل على الرابط:
   https://<username>.github.io/fast-tennis-clash/

يمكنك الآن مشاركة الرابط مع أي شخص للعب مباشرة.
